Research Info

Over the Summer of 2023 I worked on building a GANs Model that intakes a file of data related to real world drone networking data, we then trained the model to outputlifelike simulated data of drone networks. We wanted to accomplish this because there is a very limited amount of drone network data in real world application,and using our GANs model it would save a lot of money in data collection to make drone networks work better in real world scenarios.
